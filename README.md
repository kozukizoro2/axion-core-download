# AXION CORE

Free beta Windows desktop app for organizing Plex, Jellyfin and Emby media libraries.

Website: https://axioncore.app

## Features
- Smart media organizer
- Plex/Jellyfin/Emby-friendly renaming
- AniList + TMDB mapping
- Preview before apply
- FFprobe quality detection
- Undo/history
- Optional AI-assisted parsing
